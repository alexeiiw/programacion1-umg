## Flujo de trabajo

1. Abrir terminal

2. Ir a la carpeta del curso

cd ~/programacion1/programacion1-umg

3. Actualizar contenido

git pull

4. Crear laboratorio

cd 04_plantillas
./crear-lab01.sh

5. Ir al laboratorio

cd ~/programacion1/trabajos/lab01_estudiante

6. Abrir en VS Code

code .

7. Ejecutar

dotnet run
