# Programación I - UMG  
Sede Portales  

---

## 📌 Primer uso (solo una vez)

1. Abrir terminal  
2. Ir a la carpeta base:

cd ~/programacion1  

3. Clonar el repositorio:

git clone https://github.com/alexeiiw/programacion1-umg.git  

4. Ingresar al repositorio:

cd programacion1-umg  

5. Leer este archivo completo antes de continuar  

---

## 🚀 Flujo de trabajo (cada clase)

1. Ir al repositorio:

cd ~/programacion1/programacion1-umg  

---

2. Actualizar contenido:

git pull  

---

3. Crear laboratorio:

crear-lab N  

Ejemplo:

crear-lab 01  

---

4. Ir al laboratorio:

cd ~/programacion1/trabajos/labN_estudiante  

Ejemplo:

cd ~/programacion1/trabajos/lab01_estudiante  

---

5. Abrir en VS Code:

code .  

---

6. Leer instrucciones:

Abrir el archivo README.md dentro del laboratorio  

---

7. Resolver ejercicio:

Editar el archivo Program.cs  

---

8. Ejecutar:

dotnet run  

---

## 📁 Estructura del curso

01_ejemplos → Código de referencia

---

## ⚠️ Reglas importantes

- No modificar archivos del repositorio  
- Trabajar únicamente en:

~/programacion1/trabajos  

---

## 💡 Si tienes dudas

1. Revisa el README del laboratorio  
2. Revisa los ejemplos en:

~/programacion1/programacion1-umg/01_ejemplos  

3. Intenta nuevamente antes de preguntar

Desarrollar lógica de programación mediante práctica constante.
