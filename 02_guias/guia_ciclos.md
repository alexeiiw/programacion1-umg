# GUÍA 05 - CICLOS (REPETICIÓN)

## 🎯 Objetivo
Aprender a ejecutar un bloque de código varias veces sin tener que escribirlo una y otra vez.

---

## 🔄 ¿Qué es un Ciclo?
Un ciclo es una estructura que repite instrucciones mientras una condición sea **Verdadera**. Cuando la condición se vuelve **Falsa**, el ciclo se rompe y el programa continúa.



[Image of while loop flowchart]


### 1. El Ciclo `while` (Mientras...)
Se usa cuando **no sabemos exactamente cuántas veces** se repetirá algo. Primero pregunta, luego actúa.
- *Ejemplo:* Mientras la clave sea incorrecta, seguir pidiendo la clave.

### 2. El Ciclo `do-while` (Hacer, mientras...)
Es primo del `while`, pero con una diferencia clave: **siempre se ejecuta al menos una vez**. Primero actúa, luego pregunta.
- *Ejemplo:* Mostrar un menú, y al final preguntar si desea salir.

### 3. El Ciclo `for` (Para...)
Se usa cuando **sí sabemos exactamente cuántas veces** queremos repetir algo. Tiene un contador integrado.
- *Ejemplo:* Imprimir los números del 1 al 10.



[Image of for loop flowchart]

## 🧭 El Reto: El Contador Inteligente

Debes crear un programa que realice tres tareas distintas usando los tres tipos de ciclos:

1. **Con `for`:** Imprimir la tabla de multiplicar de un número que el usuario elija (del 1 al 10).
2. **Con `while`:** Crear un contador regresivo (ej. de 10 a 1) para el lanzamiento de un cohete.
3. **Con `do-while`:** Pedir un número al usuario repetidamente hasta que ingrese el número **0** para salir.
